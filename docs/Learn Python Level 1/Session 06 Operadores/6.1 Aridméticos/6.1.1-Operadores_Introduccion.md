# BootCamp Python Sesión 6 - Los Operadores

## 6.1 Operadores en Python Introducción

### ¿Qué son?

Loas `operadores` son simbolos y/o caracteres especiales que indican que se debe realizar alguna operación sobre una o más variables, estas se denominan `operandos`, el operando es la variable o el valor sobre el cual el operador ejecutara la operación.

### EJEMPLO

```python
x= 5+3
```

AQUÍ EL SIGNO DE (`+`) ES EL OPERADOR DE (`SUMA`)
`"5 + 3"` SON LOS OPERANDOS
