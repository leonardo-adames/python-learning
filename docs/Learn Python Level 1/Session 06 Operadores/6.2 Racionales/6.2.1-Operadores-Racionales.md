## 6.3 Operadores Racionales O De Comparación ==> (`==, !=, <=, >=, <, >`)

### ¿Qué son?

Étos operadores representan la comparación de valores para determinar relación lógica entre ellos.

* EL RESULTADO DE ESTAS OPERACIONES SIEMPRE SERA UN VALOR BOOLEANO ("True y/o False").

```python
#VARIABLES DE EJEMPLO:
a = 6

b = 28

#RACIONALES
print("a == b", a == b)
print("a != b", a != b)
print("a <= b", a <= b)
print("a >= b", a >= b)
print(" a <> b", a <0> b)
print("a < b", a < b)
print("a > b", a > b)
```

### OTRO EJEMPLO PRACTICO CON OPERADORES RACIONALES O DE COMPARACIÓN

1.Igualdad `— ==`

EJEMPLO:

```python
edad = 18
print(edad == 18)   # True
#Pregunta si la edad es igual a 18.
```

2.Diferente `— !=`

EJEMPLO:

```python
color = "rojo"
print(color != "azul")   # True
# Pregunta si el color no es azul.
```

3.Mayor que `>`

EJEMPLO:

```python
nota = 85
print(nota > 70)   # True
# Pregunta si la nota es mayor que 70
````

4.Menor que `<`

EJEMPLO:

```python
temperatura = 25
print(temperatura < 30)   # True
# Pregunta si la temperatura es menor que 30
```

5.Mayor o igual `>=`

EJEMPLO:

```python
saldo = 500
print(saldo >= 500)   # True
# Pregunta si el saldo es mayor o igual que 500
```

.Menor o igual `<=`

EJEMPLO:

```python
velocidad = 60
print(velocidad <= 80)   # True
# Pregunta si la velocidad es menor o igual que 80
```
