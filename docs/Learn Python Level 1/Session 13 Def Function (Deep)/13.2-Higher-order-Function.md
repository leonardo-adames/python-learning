## 13.2 Funciones de Nivel Superior en Python

### Funciones de nivel superior en Python

son funciones que tratan a otras funciones como objetos de primera clase.  Esto significa que puedes pasar funciones como argumentos a otras funciones, devolver funciones como valores y almacenar funciones en variables. Esta característica es fundamental en la programación funcional y permite escribir código más limpio, modular y reutilizable. 

### Características principales

- Pasar funciones como argumentos: Una función puede recibir otra función como parámetro. 
- Devolver funciones como resultados: Una función puede devolver otra función. 
- Almacenar funciones en variables: Puedes asignar funciones a variables, como cualquier otro valor. 

### Ejemplos comunes

- `map()`: Aplica una función a cada elemento de un iterable. 
- `filter()`: Filtra elementos de un iterable según una función condicional. 
- `sorted() con key`: Ordena una lista usando una función como criterio de orden. 
- `reduce()`: Aplica una función de forma acumulativa a los elementos de un iterable (requiere functools). 

### Ejemplo practico:

```python
# Usando filter() como función de orden superior
def es_par(numero):
    return numero % 2 == 0

numeros = [1, 2, 3, 4, 5, 6]
pares = list(filter(es_par, numeros))
print(pares)  # [2, 4, 6]   
```

### Funciones anónimas (`lambda`)

Se usan comúnmente con funciones de orden superior para definir funciones temporales sin nombre:

```python
numeros = [3, 1, 4, 1, 5, 9]
ordenados = sorted(numeros, key=lambda x: x % 2 == 0)   
print(ordenados)  # [1, 1, 3, 4, 5, 9] (pares primero)
```

### Uso en clases y decoradores

Las funciones de orden superior también se utilizan en métodos de clases y en la creación de decoradores, que añaden funcionalidades a funciones existentes sin modificar su código. 

```python
def mi_decorador(funcion):
    def wrapper():
        print("Antes de la función")
        funcion()
        print("Después de la función")
    return wrapper

@mi_decorador
def saludar():
    print("Hola!")

saludar()
# Antes de la función
# Hola!
# Después de la función
```

**Nota:** Python incluye funciones de orden superior en su biblioteca estándar (`functools`, `map`, `filter`, `reduce`, etc.) y permite crear funciones personalizadas para patrones comunes, mejorando la legibilidad y mantenibilidad del código.
