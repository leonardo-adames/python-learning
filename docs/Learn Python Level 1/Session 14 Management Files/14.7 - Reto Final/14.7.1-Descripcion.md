
# Learning Python Level 1 - Sesión 14 Management Files

## 14.7.1   Reto Final

### Descripción.

Crear un pequeño sistema de "`Log`" o una "`Agenda`" que guarde los datos en un archivo para que no se pierdan al cerrar el programa.

### Requisitos.

- Debe tener un menú con las siguientes opciones:
    - `Agregar`
    - `Mostrar`
    - `Eliminar`
    - `Salir`

- Debe tener una opción para `Agregar` un nuevo registro.
- Debe tener una opción para `Mostrar` todos los registros.
- Debe tener una opción para `Eliminar` un registro.
- Debe tener una opción para `Salir` del programa.
