## 11.3.2 `Indexación En Set` en python

### Artículos de acceso

No puedes acceder a los elementos de un conjunto consultando un índice o una clave.

Pero puedes recorrer los elementos del conjunto usando un bucle, o preguntar si hay un valor especificado en un conjunto, usando la palabra clave.`for`, y `in`

### Ejemplo

Haz un bucle por el conjunto e imprime los valores:

```python
thisset = {"apple", "banana", "cherry"}

for x in thisset:
  print(x)
```

### Ejemplo

Comprueba si "banana" está presente en el set:

```python
thisset = {"apple", "banana", "cherry"}

print("banana" in thisset)
```

### Ejemplo

Comprueba si "banana" NO está presente en el set:

```python
thisset = {"apple", "banana", "cherry"}

print("banana" not in thisset)
```

### Cambiar los elementos

Una vez creado un conjunto, no puedes cambiar sus objetos, pero sí puedes añadir nuevos.
