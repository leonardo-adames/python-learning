## 11.1.2 `Indexación De Listas` en python

### Acceso A Elementos Por Índices

Los elementos de la lista están indexados y puedes acceder a ellos consultando el número de índice siguiente:

### Ejemplo

Imprime el segundo elemento de la lista:

```python
thislist = ["apple", "banana", "cherry"]
print(thislist[1])
```

**Nota:** El primer elemento tiene índice 0.

### Indexación negativa

Indexación negativa significa empezar desde el final

`-1` se refiere al último elemento, se refiere al penúltimo elemento, etc. `-2`

### Ejemplo

Imprime el último elemento de la lista:

```python
thislist = ["apple", "banana", "cherry"]
print(thislist [-1])            # auí se imprime el ultimo elemento del indice
```

### Rango de índices

Puedes especificar un rango de índices especificando por dónde empezar y por dónde Termina el rango.

Al especificar un rango, el valor de retorno será una nueva lista con el elementos especificados.

### Ejemplo

Devuelve el tercer, cuarto y quinto ítem:

```python
thislist = ["apple", "banana", "cherry", "orange", "kiwi", "melon", "mango"]
print(thislist[2:5])
```

**Nota:** La búsqueda comenzará en el índice 2 (incluido) y terminará en el índice 5 (no incluido).

### Recuerda que el primer elemento tiene índice 0.

Al omitir el valor inicial, el rango comenzará en el primer elemento:

### Ejemplo

Este ejemplo devuelve los elementos del principio a, pero NO incluyendo, "kiwi":

```python
thislist = ["apple", "banana", "cherry", "orange", "kiwi", "melon", "mango"]
print(thislist[:4])
```

Al omitir el valor final, el rango pasará al final de la lista:

### Ejemplo

Este ejemplo devuelve los elementos de "cereza" al final:

```python
thislist = ["apple", "banana", "cherry", "orange", "kiwi", "melon", "mango"]
print(thislist[2:])
```

### Rango de índices negativos

Especifica índices negativos si quieres empezar la búsqueda desde el final de la Lista:

### Ejemplo

Este ejemplo devuelve los elementos de "naranja" (-4) a, pero NO incluye "Mango" (-1):

```python
thislist = ["apple", "banana", "cherry", "orange", "kiwi", "melon", "mango"]
print(thislist[-4:-1])
```

### Comprueba si existe un artículo

Para determinar si un elemento especificado está presente en una lista, utilice la palabra clave: `in`

### Ejemplo

Comprueba si "manzana" está presente en la lista:

```python
thislist = ["apple", "banana", "cherry"]
if "apple" in thislist:
  print("Yes, 'apple' is in the fruits list")
```
