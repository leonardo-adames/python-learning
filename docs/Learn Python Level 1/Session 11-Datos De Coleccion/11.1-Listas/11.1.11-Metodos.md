## 11.1.11 `Métodos De Listas` en python

### Métodos

Python tiene un conjunto de métodos integrados que puedes usar en listas.

| Método      | Descripción en español                                                                |
| ----------- | ------------------------------------------------------------------------------------- |
| `append()`  | Agrega un elemento al final de la lista                                               |
| `clear()`   | Elimina todos los elementos de la lista                                               |
| `copy()`    | Devuelve una copia de la lista                                                        |
| `count()`   | Devuelve el número de elementos con el valor especificado                             |
| `extend()`  | Agrega los elementos de otra lista (o cualquier iterable) al final de la lista actual |
| `index()`   | Devuelve el índice del primer elemento con el valor especificado                      |
| `insert()`  | Inserta un elemento en la posición especificada                                       |
| `pop()`     | Elimina el elemento en la posición especificada                                       |
| `remove()`  | Elimina el primer elemento con el valor especificado                                  |
| `reverse()` | Invierte el orden de los elementos de la lista                                        |
| `sort()`    | Ordena los elementos de la lista                                                      |
