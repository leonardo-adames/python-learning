## 11.2.1 `Indexación De Tuplas` en python

### Acceder a los elementos Tupla

Puedes acceder a los elementos de la tupla consultando el número de índice, en el cuadrado interior Brackets:

### Ejemplo

Imprime el segundo elemento de la tupla:

```python
thistuple = ("apple", "banana", "cherry")
print(thistuple[1])
```

**Nota:** El primer elemento tiene índice `0`.

### Indexación negativa

Indexación negativa significa empezar desde el final.

`-1` se refiere al último elemento, se refiere al penúltimo elemento, etc.`-2`

### Ejemplo

Imprime el último elemento de la tupla:

```python
thistuple = ("apple", "banana", "cherry")
print(thistuple[-1])
```

### Rango de índices

Puedes especificar un rango de índices especificando por dónde empezar y por dónde Termina el rango.

Al especificar un rango, el valor de retorno será una nueva tupla con la elementos especificados.

### Ejemplo

Devuelve el tercer, cuarto y quinto ítem:

```python
thistuple = ("apple", "banana", "cherry", "orange", "kiwi", "melon", "mango")
print(thistuple[2:5])
```

**Nota:** La búsqueda comenzará en el índice 2 (incluido) y terminará en el índice 5 (no incluido). Recuerda que el primer elemento tiene índice 0.

Al omitir el valor inicial, el rango comenzará en el primer elemento:

Este ejemplo devuelve los elementos del principio a, pero NO incluidos, "kiwi":

```python
thistuple = ("apple", "banana", "cherry", "orange", "kiwi", "melon", "mango")
print(thistuple[:4])
```

Al omitir el valor final, el rango continuará hasta el final de la tupla:

Este ejemplo devuelve los elementos de "cereza" y al final:

```python
thistuple = ("apple", "banana", "cherry", "orange", "kiwi", "melon", "mango")
print(thistuple[2:])
```

### Rango de índices negativos

Especifica índices negativos si quieres empezar la búsqueda desde el final de la Tuple:

### Ejemplo

Este ejemplo devuelve los elementos del índice `-4` (incluido) al índice `-1` (excluido)

```python
thistuple = ("apple", "banana", "cherry", "orange", "kiwi", "melon", "mango")
print(thistuple[-4:-1])
```

### Comprueba si existe un artículo

Para determinar si un elemento especificado está presente en una tupla, utilice la palabra clave:in

### Ejemplo

Comprueba si "`manzana`" está presente en la tupla:

```python
thistuple = ("apple", "banana", "cherry")
if "apple" in thistuple:
  print("Yes, 'apple' is in the fruits tuple")
```
