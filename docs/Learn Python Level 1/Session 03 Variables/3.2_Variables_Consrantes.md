## 2. CREAR VARIABLES Y  CONSTANTES

* Para crear una variable/constante en python, debemos de realizarlo de la siguiente manera:

**Variable:**
nombre_variable = <valor_a_asignar>

**Constante:**
NOMBRE_CONSTANTE = <valor_a_asignar>

### **SIEMPRE USANDO lower_snake_case PARA VARIABLES.**

### **SIEMPRE USANDO UPPER_CASE PARA CONSTANTES.**

**Ejemplos de variable:**

```python
# Variable
edad = 25               # (edad) es una variable, (25) es el valor a asignadar
nombre = "Juan Pérez"   # (nombre) es una variable, ("Juan Pérez") es el valor a asignadar
print(edad)             # Salida: 25
print(nombre)           # Salida: Juan Pérez
```

**Ejemplos de constante:**

```python
# Constante
PI = 3.14159       # (PI) es una constante, (3.14159) es el valor a asignadar
GRAVEDAD = 9.81    # (GRAVEDAD) es una constante, (9.81) es el valor a asignadar
print(PI)          # Salida: 3.14159
print(GRAVEDAD)    # Salida: 9.81
```
