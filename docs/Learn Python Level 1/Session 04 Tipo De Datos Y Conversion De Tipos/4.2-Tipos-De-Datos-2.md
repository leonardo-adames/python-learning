## 4.1 Tipos De Datos Compuestos o Colecciones en Python

### ¿Qué son?

En `Python` los `TIPOS DE DATOS COMPUESTOS` o `COLECCIONES` permiten almacenar `multiples valores dentro de una sola variable`. estos son esenciales para organizar información y trabajar con `COLECCIONES`. Junto a el grupo de `TIPOS DE DATOS SIMPLES` forman parte de los `TIPOS DEDATOS INTEGRADOS EN PYTHON`.

Python Ofrece 5 `TIPOS PRINCIPALES`:

### 1. istas --> `[list]`

En `PYTHON` las `Listas` son un tipo de datos compuesto que permite almacenar multiples tipos de elementos dentro de una `Variable` de forma ordenada.

**CARACTERISTICAS**

SUS CARACTERISTICAS SON:

* Colección Ordenada.
* Estructura Mutable (puedes cambiar y/o modificar sus elementos )
* Permiten Valores Duplicados.
* Se definen entre corchetes:

```python
# Ejemplo De Una Lista
frutas = ["Manzana", "banana", "cereza"]    #Esta es una lista
```

**CUANDO USARLAS:** 

* Cuando necesitas una colección que pueda cambiar y donde el orden importa

---

### 2.  Tuplas --> `(tuple)`

En `PYTHON` las `Tuplas` son un tipo de datos compuesto que permite almacenar multiples tipos de elementos dentro de una `Variable` de forma ordenada.

**CARACTERISTICAS**

USCARACTERISTICAS SON:

* Colección Ordenada.
* Estructura Inmutable (no se puedes cambiar y/o modificar sus elementos despues de crearse)
* Permiten Valores Duplicados.
* Se definen entre parentesis:

```python
# Ejemplo De Una Tupla
coordenadas = (10.5, 20.7, 30.9)    # Esto es una tupla
```

**CUANDO SARLAS:** Cuando quieres proteger datos sencibles (que no deben cambiar dentro del programa) como las `COORDENADAS` o `CONSTANTES`.

---
### 3. Conjuntos--> `{set}`

En `PYTHON` los `Conjuntos` son un tipo de datos compuestos que permiten almacenar multiples elementos dentro de una misma `variable` como las listas y las tuplas, solo que los `Conjuntos` no son ordenados.

**CARACTERISTICAS:**

SUS CARACTERISTICAS SON:

* Estructura No Ordenada.
* Estructura Mutable
* No permiten Duplicados.
* Se Definen Entre `llaves`:

```python
#Ejemplo De Un Conjunto
numeros_unicos = {1, 2, 3, 4, 5}    # Esto es un conjunto
```

**CUANDO USARLOS:** Cuando necesitas eliminar duplicados o hacer operaciones de teoría de conjuntos (`unión`, `intersección` y `diferencia`)

---
### 4. Conjuntos Inmutables--> `([frozenset])`

En `PYTHON` los `Conjuntos Inmutables` son la versión inmutable de los conjuntos, tipos de datos compuestos que permiten almacenar varios elementos dentro de una misma `Variable` como todos los anteriores,una vez creados, no puedes modificar y/o cambiar sus elementos.

**CARASTERISTICAS:**

SUS CARACTERISTICAS SON:

* Colección Desordenada.
* No Tienen Indices
* Es hashable:
    - se puede usar como clave de diccionario
    - se puede usar como elemento dentro de otro `set`
* se defienn de la siguiente manera:

```python
# Ejemplo De Un Conjunto Inmutable
a = frozenset([1, 2, 3, 4])
print(a)    # frozenset ({1, 2, 3, 4})

roles= frozenset(["admin", "user"])
permisos = {roles: "acceso total"}  #Válido porque frozenset es hashable
```

* Operaciones Soportadas:
    * Unión
    * Intersección
    * Diferencia
    * Diferencia Simétrica

Puede hacer todas las operaciones anteriores, igual que con `set`, sin embargo, siempre devuelve nuevos `frozenset` lo que significa que no se modifica el original.

```python
# Ejemplo De Operaciones Con frozenset
fs1 = ([1, 2, 3, 4])
fs2 = ([3, 4, 5, 6])

print(fs1 | fs2)    #Operación de Unión
print(fs1 & fs2)    #Operación de Intersección
print(fs1 - fs2)    #Operación de Diferencia
```

**CUANDO USARLOS:**

Cuando se necesita inmutabilidad y seguridad, para usarla como `clave de diccionario` o como un elemento dentro de otro `set` si es necesario

---

### 5. Diccionario--> `{ditc}`

En `PYTHON` los `Diccionarios` al igual que todos los anteriores son tipos de datos compuestos que  permiten almacenar multiples elemento dentro de una misma `Variable`, con la diferencia de que se guardan en pares de `clave: valor` muy parecido a los ficheros JSON.

**caracteristicas:**

* Colección de pares `clave--valor`.
* Estructura Mutable
* Desde `Python 3.7` matienen el orden de inserción.
* Se Definen con llaves y dos puntos:

```python
# Ejemplo De Un Diccionario
persona = {"nombre": "Leonardo", "edad": 28, "ciudad": "Santo Domingo"} 3 Este es un Diccionario
```

**CUANDO USARLOS:**

Cuando se necesita asociar valores a claves únicas, como información de un usuario o configuraciones.
