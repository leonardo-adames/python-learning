## 9.9 Sentencia De Coincidencia // `Condición Match` en python

### La sentencia de coincidencia de Python

* La sentencia se utiliza para realizar diferentes acciones basadas en distintas condiciones.`match`

* La sentencia de coincidencia de Python
En lugar de escribir muchas declaraciones, puedes usar la declaración.`if`..`else` `match`.

* La instrucción selecciona uno de los muchos bloques de código a ejecutar.match

Sintaxis

```python
match expression:
  case x:
    code block
  case y:
    code block
  case z:
    code block
```

### Así es como funciona

    La expresión se evalúa una vez.`match`

    El valor de la expresión se compara con los valores de cada .case

    Si hay una coincidencia, se ejecuta el bloque de código asociado.

    El ejemplo siguiente utiliza el número del día laborable para imprimir el nombre del día entre semana:

Ejemplo

```python
day = 4
match day:
  case 1:
    print("Monday")
  case 2:
    print("Tuesday")
  case 3:
    print("Wednesday")
  case 4:
    print("Thursday")
  case 5:
    print("Friday")
  case 6:
    print("Saturday")
  case 7:
    print("Sunday")
```

### Valor por defecto

* Usa el carácter de guion bajo _ como último Valor de caso si quieres que un bloque de código se ejecute cuando no hay otras coincidencias:

Ejemplo

```python
day = 4
match day:
  case 6:
    print("Today is Saturday")
  case 7:
    print("Today is Sunday")
  case _:
    print("Looking forward to the Weekend")
```

* El valor _ siempre coincidirá, por lo que es importante situarlo como el último caso para hacerlo compórtate como un caso por defecto.

### Valores Combine

* Utiliza el carácter de tubería | como operador de or en la evaluación de casos para comprobar para más de una coincidencia de valor en un caso:

Ejemplo

```python
day = 4
match day:
  case 1 | 2 | 3 | 4 | 5:
    print("Today is a weekday")
  case 6 | 7:
    print("I love weekends!")
```

### Declaraciones de If como Guardianes

* Puedes añadir sentencias en la evaluación del caso como comprobación adicional de condición:if

Ejemplo

```python
month = 5
day = 4
match day:
  case 1 | 2 | 3 | 4 | 5 if month == 4:
    print("A weekday in April")
  case 1 | 2 | 3 | 4 | 5 if month == 5:
    print("A weekday in May")
  case _:
    print("No match")
```
