## 9.8 Condicionales // `Introducción de pass` en python

### La Declaración De Aprobado

* if las sentencias no pueden estar vacías, pero si tú Por alguna razón, si tienes una sentencia sin contenido, pon en la sentencia para evitar que te provoque un error.ifpass

Ejemplo

```python
a = 33
b = 200

if b > a:
  pass
```

* La sentencia pass es una operación nula: no ocurre nada cuando se ejecuta. Sirve como un marcador de posición.

### ¿Por qué usar el pase?

La instrucción de paso es útil en varias situaciones:

    Cuando estás creando la estructura de código pero aún no has implementado la lógica

    Cuando una afirmación es necesaria sintácticamente pero no se requiere ninguna acción
    
    Como marcador de posición para el código futuro durante el desarrollo
    
    En funciones o clases vacías que planeas implementar más adelante

### paso en Desarrollo

* Durante el desarrollo, quizá quieras esbozar la estructura de tu programa antes de implementar los detalles. La sentencia pass te permite hacer esto sin errores de sintaxis.

Ejemplo

* Marca de posición para futuras implementaciones:

```python
age = 16

if age < 18:
  pass # TODO: Add underage logic later
else:
  print("Access granted")
```

### pasar vs Comentarios

* Un comentario es ignorado por Python, pero pasar es una sentencia real que se ejecuta (aunque no sirve de nada). Necesitas pasar donde Python espera una afirmación, no solo un comentario.

Ejemplo

* Esto causará un error (bloque de código vacío):

```python
score = 85

if score > 90:
  # This is excellent
# This will raise an IndentationError
```

Ejemplo

* Esto funciona correctamente con el paso:

```python
score = 85

if score > 90:
  pass # This is excellent
print("Score processed")
```

### Pasar con Múltiples Condiciones

* Puedes usar pass in cualquier rama de una sentencia if-elif-else.

Ejemplo

* Uso de pase en diferentes ramas:

```python
value = 50

if value < 0:
  print("Negative value")
elif value == 0:
  pass # Zero case - no action needed
else:
  print("Positive value")
```

### pasa en otros contextos

Aunque aquí nos centramos en las sentencias pass with if, también se usa comúnmente con bucles, funciones y clases.

Ejemplo

* Usando el pase con funciones:

```python
def calculate_discount(price):
  pass # TODO: Implement discount logic

# Function exists but doesn't do anything yet
```
